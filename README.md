# Mini French-English Dictionary (Beginner Go Project)

A super simple French-English dictionary in Go: enter a French word, the program gives you the English translation (if present in the list).

## Usage

1. Open this folder in your terminal.
2. Run:
   ```sh
   go run main.go
   ```
3. Type a French word (e.g., chat, chien, livreâ€¦)  
4. To quit, type `exit`.

---

To expand the list, just edit the `dictionary` variable directly in the Go code.
